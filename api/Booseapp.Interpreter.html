Table of Contents

Namespace Booseapp.Interpreter

Classes

ArrayDeclarationCommand

Declares an array (int[] or real[]) in the VariableStore.

AssignmentCommand

Assigns a value to a variable (int/real/bool) depending on what is already declared.

BooleanAssignmentCommand

Assigns a boolean expression into an existing boolean variable.

BooleanDeclarationCommand

Declares a boolean variable (optionally with an expression).

CallCommand

Calls a previously declared method and executes its body lines using the current interpreter.

CircleCommand

Draws an unfilled circle at the current cursor position.

ExpressionEvaluator

Evaluates numeric and boolean expressions for the interpreter. Supports basic arithmetic, comparisons, and logical operators.

IntDeclarationCommand

Declares an integer variable.

IntExpr

Small integer-only expression evaluator used for simple int maths. Supports + - * /, brackets, unary +/- and int variables.

MethodDefinition

Represents a parsed method definition (name, return type, parameters and body lines).

MethodTable

Stores method definitions and provides parsing for method headers.

MiniFactory

Factory that turns a single line of text into an ICommand. This is basically the interpreter's "command parser".

MiniInterpreter

Simple interpreter that runs program text line-by-line. Also handles extracting and storing method definitions.

MoveToCommand

Moves the cursor to a new X,Y position on the canvas.

PeekCommand

Reads a value from an array element into an already declared variable.

PenCommand

Sets the drawing colour using r,g,b values (clamped to 0..255).

PokeCommand

Writes a value into an array element (tries int[] first, then real[]).

RealDeclarationCommand

Declares a real (double) variable (optionally with an expression).

RectCommand

Draws an unfilled rectangle at the current cursor position.

VariableStore

Stores variables for the interpreter. Supports global and local scopes, plus int/real/bool and basic arrays.

WriteCommand

Writes text to the canvas. Supports strings, numeric expressions, boolean expressions, and simple concatenation using '+'.

Interfaces

ICommand

Base interface for all interpreter commands. Each command represents one executable line.
